🎯 About the Game

Hand Cricket Master is a mobile cricket game inspired by the classic hand cricket we all played during school days.

I built this game to recreate that simple yet exciting experience in a digital form — where strategy and a little bit of luck decide the winner.

The goal is simple:
Score as many runs as possible before getting OUT and chase targets like a true champion.

It’s easy to play, but mastering it takes focus.


🎮 How It Works

Choose a number between 1 and 6

The Computer opponent chooses its number

If both numbers match → You’re OUT

If they don’t match → You score runs

Win by scoring more than your opponent

Simple rules. Endless fun.
